# CAD — Glytch: Nemo Edition

Reference 3D assets for the 7" (17.8 cm) Glytch: Nemo Edition collectible.
These are **design-intent** files — the factory will derive the final
tool-cut STEP file during Phase 2 (CAD Development).

## Files

| File | Format | Size | Viewable on GitHub? |
|---|---|---|---|
| [`glytch-nemo-edition.stl`](./glytch-nemo-edition.stl) | Binary STL | 48 MB | ✅ GitHub renders an interactive 3D viewer inline when you click the file |
| [`glytch-nemo-edition.glb`](./glytch-nemo-edition.glb) | glTF binary | 22 MB | Download to view — [laclawclaw.com](https://laclawclaw.com) embeds this for in-browser rotation |
